# `ssh-keygen` - Make SSH key

`ssh-keygen` creates an RSA key pair and stores the public key in a public key file named `.ssh/id_rsa.pub` and a private key file named `.ssh/id_rsa`.

To make a key, you can type:

```bash
ssh-keygen -t rsa
```
