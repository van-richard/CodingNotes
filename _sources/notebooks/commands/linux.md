# List of Commands

```{Glossary}
[awk](./awk)
     manipulate text

[cat](./cat)
     print input to standard output

[cd](./cd)
     change directories

[chsh](./chsh)
     change shell

[cp](./cp)
     copy files or folders

[grep](./grep)
     search for pattern in file or folders

[head](./head)
     read from top of file to standard output

[help](./help)
     read documentation
 
[ls](./ls)
     list information about files/folders 

[man](./man)
     documentation for various functions 

[mkdir](./mkdir)
     make new directories.

[module](./module)
     load software on slurm

[mv](./mv)
     move files/folders from one place to another.

[rm](./rm)
     delete files

[rsync](./rsync)
     sync files/folders 

[scp](./scp)
     copy files/folders over ssh

[sed](./sed)
     tranform/replace text!

[seq](./seq)
     sequence of numbers

[setfacl](./setfacl)
     change file/folder permissions 

[sort](./sort)
     sorts lines

[tail](./tail)
     read from bottom of file

[tar](./tar)
     compress/uncompress files

[trap](./trap)
     signalled execution

```
