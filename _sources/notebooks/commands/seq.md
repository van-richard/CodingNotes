# `seq` - sequence of numbers

Default increment of 1.

```bash
seq FIRST INCREMENT LAST
```

This returns a sequence of numbers between 0 and 10.

```bash
seq 0 10
```

Sequence with Placeholders

```bash
seq -w 0 40
```
